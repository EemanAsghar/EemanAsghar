### 🚀 Featured projects

**🔐 [AuditML — Privacy Testing Toolkit](https://github.com/EemanAsghar/AuditML-Privacy-Toolkit)** &nbsp;·&nbsp; [`pip install auditml`](https://pypi.org/project/auditml)
Open-source library to audit ML models for privacy leakage — membership inference, attribute inference, and model inversion — with Differential Privacy (Opacus) and a **Rust (PyO3)** acceleration module (**11× faster**). 418 tests · 91% coverage.
`Python` · `PyTorch` · `Opacus` · `Rust (PyO3)`

**🛡️ [AgentProof — the deployment gate for AI agents](https://github.com/EemanAsghar/Agentproof)** &nbsp;·&nbsp; [live demo](https://agentproof-opal.vercel.app)
Continuous behavioral validation &amp; regression detection for AI agents, running natively on the **UiPath Platform**. *Traditional CI/CD tests code; AgentProof tests AI behavior.* Built for **UiPath AgentHack 2026**.
`Python` · `LangGraph` · `UiPath` · `LLM-as-judge`

**🐞 [BugCost — every bug has a price](https://github.com/EemanAsghar/BugCost)** &nbsp;·&nbsp; [live demo](https://bug-cost-nine.vercel.app/demo)
Turns a production bug into a dollar figure in seconds — federates **Sentry + Stripe + GitHub** with a single **Coral SQL** query and **Claude** to pinpoint the bug, the commit that caused it, and the revenue lost. 45 minutes of manual correlation → 9 seconds.
`Next.js` · `TypeScript` · `Coral SQL` · `Claude`
